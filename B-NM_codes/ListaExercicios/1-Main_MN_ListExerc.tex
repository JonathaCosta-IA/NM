	\begin{abstract} 
Este documento reúne uma lista de exercícios de Métodos Numéricos para os cursos de Engenharia de Controle e Automação e Engenharia Mecânica do IFCE \textit{campus} Maracanaú,
alinhada ao Plano de Unidade Didática (PUD). São trabalhados temas centrais como análise de erros, métodos para equações não lineares, sistemas lineares, interpolação,
ajuste de curvas, integração e diferenciação numérica. Os exercícios são direcionados para aplicações reais de engenharia, com ênfase em modelagem,
simulação e avaliação do desempenho dos algoritmos. O foco é direto: consolidar rigor analítico e formar o aluno para implementar métodos numéricos de forma eficiente, 
visando aplicações computacionais mais avançadas.	
\end{abstract}
	
\end{frontmatter}
\section*{Instruções gerais}

Os exercícios devem ser resolvidos integrando conceitos de matemática, linguagens de programação e fundamentos de engenharia.
Esta lista de exercícios de Métodos Numéricos (MN) está organizada por áreas de conhecimento, conforme as seções listadas a seguir:

\begin{itemize}
	\item Seção \ref{sec:conversão} - conversão de números conforme norma IEEE 754/2008;
	\item Seção \ref{sec:diferenciação} - utilização de derivada numérica
	\item Seção \ref{sec:integração} - utilização de integração numérica 
	\item Seção \ref{sec:ajusteCurvas} - utilização de polinômios interpoladores e ajuste  de curvas
	\item Seção \ref{sec:SolSistLineares} - utilização de métodos convencionais e iterativos
	\item Seção \ref{sec:SolSistNaoLineares} - utilização de métodos para solução de sitemas de equação não lineares
	\item Seção \ref{sec:ExercicioIntetradosProjetos} - utilização integrada de MNs em aplicações de engenharia

\end{itemize}
Para fins de consulta e modelagem basilar, soluções alternativas estão disponíveis no repositório do professor pelo endereço: 
\url{https://github.com/JonathaCosta-IA/NM/tree/main/B-NM_codes}. Utilize-as para otimizar suas soluções quando necessário.
